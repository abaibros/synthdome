# SynthDome 웹사이트 파일 구조

## 📁 주요 파일

### 1. index.html (40KB)
- 메인 HTML 파일
- 모든 섹션 포함 (Hero, 문제점, Why Exists, 솔루션, 대형 구조, 기술, 경험 흐름, 차별점, 연락처)
- 완전한 한글화
- SVG 아이콘과 이모지 포함

### 2. styles.css (25KB)
- 전체 스타일시트
- 반응형 디자인 (데스크톱, 태블릿, 모바일)
- 그라데이션 효과
- 애니메이션
- Glassmorphism 효과

### 3. assets/script.js (5KB)
- JavaScript 인터랙션
- 스크롤 애니메이션
- 네비게이션 활성화
- 카운터 애니메이션
- Intersection Observer

## 🎨 디자인 특징

- **프리미엄 디자인**: Apple/Stripe 수준
- **완전 한글화**: 모든 텍스트 한국어
- **반응형**: 모든 기기 완벽 대응
- **애니메이션**: 부드러운 전환 효과
- **아이콘**: SVG + 이모지 조합
